## ✍️ Adding Blog Posts

All blog content is managed via `post.json`. Add a new object to the `posts` array:

```json
{
  "id": "7",
  "slug": "your-post-slug",
  "title": "Your Post Title",
  "date": "7 May 2026",
  "dateiso": "2026-05-07",
  "category": "Security Guide",
  "emoji": "🔑",
  "featured": false,
  "readtime": "5 min read",
  "author": "Security Research Team",
  "authorTitle": "BestPasswordGenerator.org",
  "excerpt": "One or two sentence summary for the blog card.",
  "tags": ["tag1", "tag2"],
  "content": "<p>Full article HTML here. Use semantic HTML — h2, h3, p, ul, strong etc.</p>"
}
```

**Categories available:** Security Guide · Statistics · Reviews · Threat Intelligence · Explainer

Set `"featured": true` on one post only — it appears as the hero article on the blog page.

## 🔗 Affiliate Slots

Three affiliate slots in `index.html` (search for `AFFILIATE SLOT`):

```html
<!-- AFFILIATE SLOT 1: Password Manager — e.g. 1Password, NordPass -->
<!-- AFFILIATE SLOT 2: VPN Service — e.g. NordVPN, ExpressVPN -->
<!-- AFFILIATE SLOT 3: Identity Protection — e.g. Aura, LifeLock -->
```

One sidebar slot in `blog.html` (search for `AFFILIATE SLOT — Sidebar`).

Replace `href="#affiliate-N"` with your tracking URL and add `rel="nofollow sponsored"`.  
Ensure affiliate disclosure text above each slot remains visible.
